Exercise 37: Count Word Frequency in a Sentence

## Objective: 
Count how often words appear in a sentence using a dictionary.

## Task:
    • Given the sentence: ”Python is great and Python is fun”, count how many times each
    word appears.

## Expected Output:
    {'Python': 2, 'is': 2, 'great': 1, 'and': 1, 'fun': 1}
