Exercise 59: Write a Sales Record to a File

## Objective: 
    Store sales records in a text file.

## Task:
    • Prompt the user for customer name, item, and price.
    • Write a formatted record (e.g., “Customer: John, Item: Laptop, Price: $1200”) to
    sales.txt.

## Expected Output:
    Enter customer name: John
    Enter item: Laptop
    Enter price: 1200
    Record written to sales.txt
