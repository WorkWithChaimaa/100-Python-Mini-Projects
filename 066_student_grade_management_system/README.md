Project 66: Student Grade Management System

## Objective:
    Build a system to store, update, and retrieve student grades.

## Task:
    • Create a dictionary where student names are keys and their grades are values.
    • Allow users to: Add a new student. Update an existing grade. Retrieve a student’s
    grade. Display all students.

## Expected Output:
1. Add Student
2. Update Grade
3. Get Grade
4. Display All
5. Exit
Enter choice: 1
Enter student name: Alice
Enter grade: A
Student added successfully.
